# Cutting Mat Wallpaper Generator

Generate desktop/phone/tablet wallpapers styled like a cutting mat: gridlines,
diagonal guide, ruler ticks, and a subtle sheen. Pick a base color, choose a
resolution, optionally add a center headline, vertical side label, and
bottom-right blurb, then export as PNG.

## Development

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

Everything renders client-side via `<canvas>` — no backend required.
